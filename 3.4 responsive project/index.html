CSS selectors

Selectors are the part which come before the {}

Selector types

  • Type selectors
  • Descendant selectors
  • Multiple selectors
  • Combining selectors
  • Combining selectors
  • Pseudo selectors

Type

Type selectors are the most basic selector

Selector Example Description
Type h1{} Targets simple html elements

Descendant

Refers to nested selectors. This does not refer to direct descendants. Can include multiple levels down as well

Selector Example Description
Descendant section h1{} Targets nested html elements (direct and ancestor relationship)

Multiple selectors

When wanting to apply the same properties to multiple selectors.

Selector Example Description
Multiple td,th {border: 1px solid burlywood;} The different selectors are separated by a comma

Combining selectors

You combine selectors to show a SPECIFIC relationship. There is no space between the selectors to differentiate it from descendant selectors.

Selector Example Description
Combining selectors p.intro{} This styling only applies to a <p> element with the class of .intro

Attribute selectors

Targeting elements with certain attributes

Selector Example Description
Attribute selectors input[type="submit"]{} The selector targets input elements where the type attribute is set to submit

Pseudo selectors

Special states of classes, elements & ids eg:
  • mouse hover
  • visited/unvisited links
  • element when in/out of focus

Selector Example Description
Pseudo selectors div:hover{} This styling only applies when mouse is hovered over the element
